dop / development operations pipeline

development operations pipeline

collection of tools and conventions for
local development and remote deployment

  • View On GitHub

Description

This framework provides a common structure for projects and applications which may be developed locally and deployed remotely on a server. It provides flexible components which can be mixed and matched to only contain the specific services needed.

Tools (essentials)

  • packer.io
  • vagrant
  • virtualbox
  • chef
  • librarian
  • knife-solo
  • knife-solo_data_bag
  • git

Components

  • dop / blueprint for a dop project
  • dop_base / base setup of dop framework
  • dop_mysql / setup of mysql as dop component
  • dop_main / wrapper cookbook for multi-application installations
  • dop_magento / setup of magento with dop component
  • dop_gitlab / setup of dedicated git hosting with dop component
  • dop_expressionengine / generic expressionengine setup
